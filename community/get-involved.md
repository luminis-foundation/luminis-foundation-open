# Get Involved

Luminis Foundation is building sovereign, ecologically-situated AI in the open. We welcome contributors who share our commitment to privacy, decentralization, and ecological alignment.

## Ways to Contribute

### Research
- Propose research ideas using the [Research Idea](https://github.com/luminis-foundation/luminis-foundation-open/issues/new?template=research-idea.yml) issue template
- Report field observations using the [Field Observation](https://github.com/luminis-foundation/luminis-foundation-open/issues/new?template=field-observation.yml) template
- Review and discuss published research

### Documentation
- Improve existing documentation
- Translate materials
- Help write educational content

### Technical
- Contribute to sensor specifications and deployment protocols
- Review and improve repository structure
- Suggest tools and infrastructure

### Community
- Share the project with aligned researchers and builders
- Provide feedback on mission and direction
- Connect us with relevant work in your field

## How to Start

1. Read the [foundational paper](https://doi.org/10.5281/zenodo.20143391)
2. Review the [governance](../governance/) documents
3. Browse [open issues](https://github.com/luminis-foundation/luminis-foundation-open/issues)
4. Open an issue or pull request
